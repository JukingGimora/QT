本开源股票数据收集分析脚本旨在方便广大股票数据分析者，为量化交易贡献一份力量。
按先后顺序整体分为五部分：
1）TestStockList:第一次全量获取上市公司名称和代码，之后增量获取要插入或更新数据；
2）TestCompanyInfo:第一次手动按步长获取1）所抓取全量上市公司，之后对1）增量抓取的上市公司进行基本信息获取（受反爬影响，阶段1,2未能合并等）
3）TestStockInfo:手动按步长获取多数每日股票数据信息，再增量获取遗漏的每日股票数据信息，判断是否有垃圾数据再更新（受反爬影响，有时需手动中断）
4) TestSql:对库里数据进行基本面分析，如统计特定时间段平均收盘价、平均交易量等（待完善）
5）TestScript:对库里数据进行学习拟合分析，如用lstm预测价格走势等（待完善）
